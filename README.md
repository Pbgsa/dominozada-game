# Dominozda Game (prototipo)


Protótipo de jogo de dominó digital com ambientes personalizáveis e ambientação sonora imersiva.

## Estrutura

- `assets/`: Recursos brutos (imagens, sons)
- `godot/dominozada/`: Código-fonte do projeto Godot
  - `scenes/`: Cenas do jogo
  - `scripts/`: Lógica em GDScript
  - `textures/` e `sounds/`: Assets processados

## Como Rodar

1. Abra a Godot Engine
2. Selecione o projeto na pasta `godot/dominozada/`
3. Clique em **Run**
